WP Post Author – Post author box | Author social icons | Author bio - Widgets and Shortcodes


== Description ==

**WP Post Author** – Post author box, social contact icons, author bio section, widgets as well as author shortcodes for WordPress single post. Basically, the plugin displays will author image, author name, author role, author email address and author short bio under post content. There's is also an availability of hide and show specific part of author information for the specific site.


### The Plugin Features:
* **Post author box** - WP Post Author for single post and single author page.
* Title - Section title.
* Alignment - Left, Right or Center.
* Profile image layout - Square or Round.
* Show author role - Yes or No.
* Show author email - Yes or No.

* **Widgets** - WP Post Author for single post and single author page.
* Title - Widget title.
* Widget content Alignment - Left, Right or Center.
* Profile image layout - Square or Round.
* Show author role - Yes or No.
* Show author email - Yes or No.

* **Shortcodes** - [wp-post-author].
* title - Any text string.
* align - left, right or center.
* image-layout - square or round.
* show-role - true or false.
* show-email - true or false.

* **Other options** - For both, widget and shortcode.
* Facebook - Display if available (Field added from plugin to user profile contact info).
* Twitter - Display if available (Field added from plugin to user profile contact info).
* LinkedIn - Display if available (Field added from plugin to user profile contact info).
* Website - Display if available.

== Installation ==
* Download to your plugin directory or simply install via WordPress admin interface.
* Activate.
* WP Post Author widget will be available for WordPress admin widget page.
* Shortcode [wp-post-author] will be workable.
* If configuration instructions needed, go to WP Post Author admin menu and follow the methods.
