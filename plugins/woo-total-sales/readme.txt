This plugin facilitates extended overview of WooCommerce dashboard sales status. Additionally, it can be used to display total sales of any specific product. 

== Description ==

Woo Total Sales – This plugin facilitates extended overview of WooCommerce dashboard sales status. Additionally, it displays total sales of specific product on shop-archives and respective single product page (Backend and Frontend). 

In the similar manner, the plugin also features backend sales overview for each of the product. You can find a "Total Sales" metabox for the individual product on which sold item number, total revenues from the product. Extended features on the meta box are as follows: 
- Admin has an option to show/hide total sales for the individual product.
- Moreover, there's an another option for the custom total sales number.

Further more, Its has its own dashborad to overview and backend options to the WooCommerce products setting to edit, to control and update "total sales" texts ( singular and plural ), "bar-chart" color and that texts too.

Contribution link: https://github.com/shshanker/woo-total-sales


### The Plugin Features:

* **New feature** - Powered with `[awts-total-sales product_id="0" include_setting="true/false"]` **shortcodes**. 

 * **awts-total-sales** - Main shortcodes

 * **product_id** - Intended valid product id

 * **include_setting** - Option to include/exclude default Woo Totals Sales visibility setting. Default is "true". 

* **Extended WooCommerce sales status** from WooCommerce admin menu.

* **Monthly sales overview** on different panel.

* Show/hide total sales **option for specific product metabox**.

* Option to **enter custom total sales number** for specific product metabox.

* Visible **only on single product** page (frontend).

* Visible **only on backend**.

* **Total sales dashboard** under WooCommerce setting menu.

* **Total sales text** - Enables WooCommerce shop manager to display total sales with product price with singular and plural forms of the counting number.

* **Icon** - For some sort of UI/UX aspect, it would automatically add the Bar-Chart icon in-front of total sales text.

* **Zero Sales** - This functionality has been disabled for the 0-sold products and would be automatically added if the total sales of the specific product exceed and equals to 1.

* **Admin Setting** - The plugin has a facility to edit and update the "Total Sales" texts under WooCommerce product general setting section. 

* **Colorful icon and texts** - And lastly you can set distinct bar-icon color and texts color if you really want to highlight or match the current theme UI/UX with total sales portion, other wise display as it is. 

* **You may also like** - [Woo Set Price Note (Units, Offers, Editions)](https://wordpress.org/plugins/woo-set-price-note/). 



== Installation ==
* Download to your plugin directory or simply install via WordPress admin interface.
* Activate.
* Go to WooCommerce admin setting menu and find Total Sales tab/panel.
* Set your texts and set color as described.
* Set visibility.
* Save your changes.
* Enjoy the plugin.
